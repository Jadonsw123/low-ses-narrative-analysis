# NLP for Social Science: Low-SES Student Narrative Analysis

## Project Overview

This project analyzes low-SES student narratives through three theoretical frameworks:
- **Bourdieu's Capital Theory** (economic, social, cultural, symbolic capital)
- **Critical Pedagogy** (student agency, consciousness of barriers)
- **Ecological Systems Theory** (micro, meso, exo, macro, chronosystem levels)

Narratives are processed through GPT-4 for semantic analysis and automatically coded against all frameworks.

## Quick Start

### Step 1: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 2: Set OpenAI API Key
```bash
# Windows (PowerShell)
$env:OPENAI_API_KEY = "your-api-key-here"

# Linux/Mac
export OPENAI_API_KEY="your-api-key-here"
```

### Step 3: Run Analysis
```bash
# Process narratives through GPT-4 (generates checkpoint files every 50 narratives)
python gpt4_three_theory_analyzer.py

# Generate visualizations from results
python visualize_results.py
```

### Step 4: Review Results
All outputs are saved to `results/` directory:
- `.json` files: Raw analysis data
- `.png` files: Publication-quality visualizations

## Workflow Details

### Input Data
- **Low_SES_Collected_dataset.csv**: Student narratives from Nahed's repository
- **gpt4_coding_prompt.md**: GPT-4 system prompt for consistent coding

### Processing Pipeline

1. **gpt4_three_theory_analyzer.py**
   - Loads narratives from CSV
   - Sends each narrative to GPT-4 with structured coding prompt
   - Codes each narrative against all three theoretical frameworks
   - Saves results in batches (checkpoints every 50 narratives)
   - Final output: `gpt4_raw_results.json`

2. **visualize_results.py**
   - Merges checkpoint files and raw results
   - Generates 6 publication-quality visualizations:
     * Agency vs. Consciousness scatter plot
     * Capital distribution heatmap
     * Ecological systems bar chart
     * Capital profile distributions
     * Agency levels distribution
     * Consciousness levels distribution

### Output Structure

## Key Features

### Theoretical Frameworks

**Bourdieu's Capital** (4 forms, 3 categories each)
- Economic, Social, Cultural, Symbolic
- Coded as: Asset, Deficit, or Absent

**Critical Pedagogy**
- Student Agency: High / Moderate / Low
- Consciousness of Barriers: High / Moderate / Low

**Ecological Systems**
- Dominant Level: Microsystem, Mesosystem, Exosystem, Macrosystem, Chronosystem

### Visualization Output

All graphs include:
- Raw counts and percentages
- Publication-ready styling (300 DPI, professional colors)
- Academic font sizing and formatting
- Clear legends and axis labels

## Customization

### Modify Coding Prompt
Edit `gpt4_coding_prompt.md` to adjust how narratives are analyzed.

### Adjust Visualization Styling
Edit `visualize_results.py` to change colors, sizes, or chart types.
